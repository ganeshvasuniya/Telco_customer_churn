# 📌 Overview

Customer churn is one of the biggest challenges in the telecom industry. Acquiring a new customer costs significantly more than retaining an existing one.

This project builds a **Machine Learning Classification Model** that predicts whether a customer is likely to leave a telecom company. By identifying high-risk customers early, businesses can proactively offer retention strategies and reduce revenue loss.

# 📈 Key Insights

- Month-to-month customers churn the most.
- Higher monthly charges increase churn probability.
- Fiber optic customers show higher churn.
- Electronic check users have the highest churn rate.
- Customers with longer tenure are much less likely to churn.
- More subscribed services reduce churn.
